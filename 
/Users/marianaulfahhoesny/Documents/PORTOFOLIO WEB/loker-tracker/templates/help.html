    Memulai Menggunakan Loker Tracker

    Panduan dasar untuk pengguna baru

    1 Halaman Dashboard

    Dashboard adalah halaman utama yang menampilkan ringkasan semua lamaran kerja Anda. Di sini Anda dapat melihat:

    • Statistik total lamaran, status, dan tingkat keberhasilan
    • Grafik visualisasi data lamaran
    • Lamaran terbaru yang ditambahkan
    • Aksi cepat untuk menambah lamaran baru
    2 Daftar Lamaran

    Halaman ini menampilkan semua data lamaran kerja dalam bentuk tabel yang dapat di-filter dan diurutkan.

    • Gunakan filter untuk menampilkan data berdasarkan status, sumber, atau rentang tanggal
    • Klik pada badge status untuk mengubah status langsung
    • Edit atau hapus lamaran dengan tombol aksi di kolom terakhir
    • Lihat bukti lamaran dengan mengklik link "Lihat" atau "Screenshot"
    3 Menambah Lamaran Baru

    Klik tombol "Tambah Lamaran" untuk menambahkan data lamaran kerja baru.

    • Isi semua field yang wajib: Posisi, Perusahaan, dan Tanggal Apply
    • Pilih sumber lamaran dari dropdown yang tersedia
    • Upload screenshot atau bukti lamaran (opsional)
    • Simpan data untuk melihatnya di dashboard dan daftar lamaran
    Cara Menambah Lamaran Kerja

    Panduan detail mengisi form tambah lamaran

    Field Wajib
    Posisi/Jabatan: Nama posisi yang dilamar (contoh: "Software Engineer", "Marketing Specialist")
    Nama Perusahaan: Nama lengkap perusahaan (contoh: "PT. Teknologi Indonesia", "Startup ABC")
    Tanggal Apply: Tanggal saat Anda melamar (default: tanggal hari ini)
    Field Opsional
    Lokasi: Lokasi kerja (contoh: "Jakarta Selatan", "Remote", "Surabaya")
    Sumber: Platform dimana Anda menemukan lowongan (LinkedIn, JobStreet, Indeed, dll)
    Link Bukti: URL ke halaman apply atau email konfirmasi
    Screenshot: Upload gambar bukti lamaran (format: JPG, PNG, max 5MB)
    Tips Mengisi Data
    • Gunakan nama posisi yang jelas dan mudah dipahami
    • Tulis nama perusahaan dengan lengkap dan benar
    • Selalu simpan screenshot atau bukti apply untuk referensi
    • Pilih sumber yang tepat untuk tracking performa platform
    Mengelola Status Lamaran

    Cara mengubah dan memantau status lamaran kerja

    Status yang Tersedia
    Terdaftar Lamaran baru saja dikirim, belum ada respon
    Interview Dipesan untuk interview atau panggilan telepon
    Tes Diundang untuk mengikuti tes/assessment
    Diterima Lamaran diterima dan mendapat offer
    Tidak Diterima Lamaran ditolak atau tidak ada respon
    Cara Mengubah Status
    1. Klik pada badge status di tabel "Daftar Lamaran"
    2. Dropdown menu akan muncul dengan opsi status
    3. Klik status baru yang diinginkan
    4. Status akan langsung terupdate secara otomatis
    Tips Monitoring Status
    • Update status secara berkala untuk tracking yang akurat
    • Gunakan halaman "Laporan" untuk melihat statistik
    • Dashboard menampilkan ringkasan status dalam bentuk grafik
    • Status "Diterima" akan meningkatkan persentase keberhasilan
    Fitur Pencarian dan Filter

    Cara mencari dan memfilter data lamaran kerja

    class="fas fa-filter me-2">Opsi Filter
    Cari Posisi/Perusahaan: Ketik kata kunci untuk mencari posisi atau nama perusahaan tertentu
    Status: Filter berdasarkan status lamaran (Terdaftar, Interview, Tes, Diterima, Tidak Diterima)
    Sumber: Filter berdasarkan platform tempat melamar (LinkedIn, JobStreet, dll)
    Rentang Tanggal: Filter berdasarkan periode apply (tanggal dari dan sampai)
    Tips Pencarian
    • Gunakan kata kunci yang spesifik untuk hasil yang akurat
    • Kombinasi filter akan memberikan hasil yang lebih tepat
    • Filter tanggal berguna untuk analisis per periode
    • Hasil filter dapat di-export untuk laporan
    Reset Filter

    Klik tombol "Reset" untuk menghapus semua filter dan menampilkan semua data.

    Export Data dan Laporan

    Cara mengunduh data dalam format PDF dan Excel

    Export PDF

    Format PDF cocok untuk laporan formal dan presentasi:

    • Data diformat rapi dalam tabel
    • Include statistik dan grafik
    • Sudah siap untuk dicetak
    • Tersedia di halaman "Laporan & Export"
    Export Excel

    Format Excel untuk analisis dan manipulasi data lebih lanjut:

    • Data dalam format spreadsheet
    • Dapat di-filter dan diurutkan
    • Cocok untuk analisis menggunakan Excel
    • Dapat diintegrasikan dengan tools lain
    Filter Data Export

    Sebelum export, Anda dapat mengatur filter:

    • Rentang tanggal untuk periode tertentu
    • Status tertentu untuk analisis status
    • Data yang sudah di-filter akan ter-export
    • Gunakan filter di halaman "Daftar Lamaran" terlebih dahulu
    Troubleshooting

    Solusi untuk masalah umum yang mungkin terjadi

    Data tidak muncul di dashboard

    Penyebab: Database kosong atau ada masalah koneksi

    Solusi:

    • Tambahkan data lamaran baru melalui tombol "Tambah Lamaran"
    • Refresh halaman untuk memuat ulang data
    • Periksa koneksi internet Anda
    Gagal upload screenshot

    Penyebab: Format file tidak didukung atau ukuran terlalu besar

    Solusi:

    • Gunakan format JPG, PNG, atau GIF
    • Pastikan ukuran file kurang dari 5MB
    • Kompres gambar jika perlu
    Status tidak berubah saat diklik

    Penyebab: JavaScript tidak aktif atau ada error

    Solusi:

    • Refresh halaman dan coba lagi
    • Pastikan JavaScript diaktifkan di browser
    • Coba gunakan browser lain
    Export PDF/Excel tidak berhasil

    Penyebab: Tidak ada data atau ada error server

    Solusi:

    • Pastikan ada data lamaran di database
    • Coba filter data terlebih dahulu
    • Periksa koneksi internet
    FAQ (Frequently Asked Questions)

    Jawaban untuk pertanyaan yang sering diajukan

    Apakah data saya aman?

    Ya, semua data tersimpan secara lokal di perangkat Anda. Kami tidak mengumpulkan atau menyimpan data pribadi Anda di server.

    Bisakah saya backup data?

    Ya, Anda dapat menggunakan fitur export untuk membuat backup data dalam format Excel. Simpan file backup di tempat yang aman.

    Bagaimana jika saya lupa password?

    Aplikasi ini tidak menggunakan sistem user login yang kompleks. Data tersimpan lokal dan tidak terikat dengan akun.

    Bisakah saya menggunakan di multiple device?

    Data tersimpan lokal, jadi untuk menggunakan di device lain, Anda perlu memindahkan file database atau menggunakan fitur export/import.

    Apakah ada batasan jumlah lamaran?

    Tidak ada batasan jumlah lamaran yang bisa Anda input. Namun semakin banyak data, semakin besar ukuran file database.

    Bagaimana cara menghapus semua data?

    Untuk keamanan, fitur hapus semua data tidak tersedia di interface. Anda dapat menghapus data satu per satu atau reset database dari menu pengaturan.
