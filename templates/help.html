    Memulai dengan Loker Tracker

    Panduan dasar untuk menggunakan aplikasi

    Selamat Datang di Loker Tracker!

    Aplikasi Loker Tracker dirancang untuk membantu Anda mengelola dan melacak lamaran kerja dengan mudah dan terorganisir.

    1
    Login ke Aplikasi

    Masukkan username dan password yang telah diberikan untuk mengakses dashboard.

    2
    Jelajahi Dashboard

    Dashboard menampilkan ringkasan data lamaran kerja Anda, termasuk statistik dan aktivitas terbaru.

    3
    Mulai Menambah Lamaran

    Klik menu "Tambah Lamaran" untuk memasukkan data lamaran kerja pertama Anda.

    Tips: Gunakan sidebar navigation untuk berpindah antar halaman dengan mudah.
    Cara Menambah Lamaran Kerja

    Panduan step-by-step untuk menambahkan lamaran kerja baru

    Langkah-langkah Menambah Lamaran:
    Informasi Perusahaan
    • Nama Perusahaan: Masukkan nama lengkap perusahaan
    • Posisi: Jabatan yang dilamar
    • Lokasi: Kota atau lokasi kerja
    • Alamat: Alamat lengkap perusahaan (opsional)
    Sumber Informasi
    • LinkedIn: Dari platform LinkedIn
    • Instagram: Postingan di Instagram
    • JobStreet: Portal karir JobStreet
    • Teman: Rekomendasi dari teman
    • Lainnya: Sumber lainnya
    Upload Bukti
    • Bukti Lamaran: Link ke email konfirmasi atau portal
    • Screenshot: Upload gambar bukti lamaran
    Informasi Tambahan
    • Tanggal Lamar: Tanggal mengirim lamaran
    • Catatan: Catatan tambahan (opsional)
    Tips: Pastikan informasi yang dimasukkan akurat dan lengkap untuk memudahkan tracking.
    Mengelola Lamaran Kerja

    Cara melihat, edit, dan menghapus data lamaran

    Fitur-fitur Manajemen:
    Pencarian & Filter

    Gunakan filter berdasarkan status, sumber, dan rentang tanggal untuk mencari lamaran tertentu.

    Edit Data

    Klik tombol edit untuk memperbarui informasi lamaran yang sudah ada.

    Hapus Data

    Gunakan tombol hapus untuk menghapus data lamaran yang tidak diperlukan lagi.

    Sorting

    Klik header kolom untuk mengurutkan data berdasarkan kolom tersebut.

    Perhatian: Data yang dihapus tidak dapat dikembalikan. Pastikan sebelum menghapus.
    Melacak Status Lamaran

    Cara mengupdate dan melacak status lamaran kerja

    Status yang Tersedia:
    Terdaftar

    Lamaran sudah dikirim dan menunggu respon

    Interview

    Diundang untuk interview

    Tes

    Mengikuti tes kemampuan atau psikotes

    Diterima

    Lamaran diterima dan mendapat pekerjaan

    Tidak Diterima

    Lamaran ditolak atau tidak melanjutkan proses

    Cara Mengupdate Status:
    1. Klik pada badge status di tabel
    2. Pilih status baru dari dropdown yang muncul
    3. Status akan terupdate otomatis
    Informasi: Perubahan status akan tersimpan otomatis dan dapat dilihat di dashboard.
    Export Data Lamaran

    Cara mengekspor data lamaran ke file Excel atau PDF

    Format Export yang Tersedia:
    Excel (.xlsx)

    Format Excel dengan semua data lengkap, cocok untuk analisis dan pengolah data.

    Menggunakan fitur filter dan formula Excel
    PDF (.pdf)

    Format PDF yang siap cetak, cocok untuk laporan dan dokumentasi.

    Layout profesional dengan header dan footer
    Langkah-langkah Export:
    1. Buka halaman "Daftar Lamaran"
    2. Klik tombol "Export Excel" atau "Export PDF"
    3. File akan otomatis terdownload
    4. Buka file dengan aplikasi yang sesuai
    Tips: Export data secara berkala untuk backup dan analisis performa lamaran Anda.
    Troubleshooting & Masalah Umum

    Solusi untuk masalah yang mungkin terjadi

    Masalah Umum & Solusi:
    Tidak bisa login

    Solusi:

    • Pastikan username dan password benar
    • Periksa koneksi internet
    • Clear cache browser dan coba lagi
    Upload bukti gagal

    Solusi:

    • Periksa ukuran file (maksimal 5MB)
    • Gunakan format JPG, PNG, atau PDF
    • Pastikan koneksi internet stabil
    Data tidak tampil

    Solusi:

    • Refresh halaman (F5 atau Ctrl+R)
    • Periksa filter yang aktif
    • Coba gunakan browser berbeda
    Export tidak berhasil

    Solusi:

    • Pastikan ada data untuk diexport
    • Tutup popup blocker
    • Coba dengan browser yang berbeda
    Masih bermasalah? Hubungi support untuk bantuan lebih lanjut.
    Frequently Asked Questions

    Jawaban untuk pertanyaan yang sering diajukan

    Apakah data saya aman?

    Ya, semua data Anda tersimpan dengan aman di server yang terlindungi. Kami menggunakan enkripsi untuk melindungi informasi pribadi Anda.

    Berapa banyak lamaran yang bisa ditambahkan?

    Tidak ada batasan jumlah lamaran yang bisa ditambahkan. Anda bisa menyimpan semua data lamaran kerja tanpa batasan.

    Apakah bisa akses dariHP?

    Ya, aplikasi ini responsive dan bisa diakses dari berbagai perangkat termasuk smartphone dan tablet.

    Bagaimana cara backup data?

    Anda bisa melakukan backup dengan mengekspor data ke format Excel atau PDF secara berkala melalui menu Daftar Lamaran.

    Apakah ada fitur notifikasi?

    Saat ini aplikasi menampilkan status secara real-time. Anda bisa mengatur reminder di kalender pribadi untuk follow up lamaran.
